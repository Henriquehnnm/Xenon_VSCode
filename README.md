## About The Project

**Xenon** is designed for developers who love dark interfaces but still want a splash of vibrant, carefully-selected color. It provides a high-contrast, low-distraction environment with a focus on readability and modern aesthetics.

## Features / Variations

-   **Xenon**: The main theme, offering a balanced and modern dark experience.
-   **Xenon - One Dark Remap**: A variation with syntax highlighting inspired by the popular One Dark theme.
-   **Xenon - Monokai Remap**: A variation with syntax highlighting inspired by the classic Monokai theme.
-   **Xenon - GitHub Dark Remap**: A variation with syntax highlighting inspired by the GitHub Dark theme.
-   **Xenon - Solarized Remap**: A variation with syntax highlighting inspired by the Solarized color palette.

## Installation

1.  Open **Visual Studio Code**.
2.  Navigate to the **Extensions** view by clicking the icon in the sidebar or pressing `Ctrl+Shift+X`.
3.  Search for **"Xenon"**.
4.  Click the `Install` button.
5.  Go to `File > Preferences > Color Theme` (or press `Ctrl+K Ctrl+T`) and select your preferred Xenon variant.
